Mini Search Engine
A simple Python-based Mini Search Engine that allows users to search for keywords inside text files and retrieve matching results quickly. This project demonstrates file handling, string matching, indexing concepts, and basic search algorithms using Python.

Features
- Search keywords from multiple text files
- Displays matched file names and lines
- Fast keyword lookup
- Case-insensitive searching
- Simple command-line interface
- Lightweight and beginner-friendly

Technologies Used
- Python 3
- File Handling
- Dictionaries
- String Processing

How It Works
1. The program scans text files inside the data folder.
2. User enters a search keyword.
3. The engine checks all files for matching words.
4. Matching results are displayed with file names and content.
